DMJsJvm Hello World example

Open the javascript console. There is a log entry: "Hello World!" (page refresh may be required if console was not already open)

The java source code: 

package bz.davide.dmjsjvm_demo;

public class HelloWorld
{
   public static void main(String[] args)
   {
      System.out.println("Hello World!");
   }
}
